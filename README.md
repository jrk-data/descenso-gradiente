# Proyecto: Descenso de Gradiente

Este repositorio contiene los materiales y laboratorio para el análisis de datos y aplicación del algoritmo de Descenso de Gradiente.

## Estructura del Proyecto

El proyecto está organizado en las siguientes secciones principales:

### 1. `App/` - Laboratorio Shiny
Esta carpeta contiene la aplicación web interactiva ("laboratorio") desarrollada en R Shiny. 
- **Propósito**: Permitir la interacción dinámica con el modelo y el algoritmo.
- **Estado**: Esta aplicación fue subida y desplegada en shinyapps.io.
- **Archivos Clave**: `app.R` (Lógica de la aplicación), `kc_house_data.csv` (Dataset utilizado).

### 2. Análisis Exploratorio de Datos (EDA)
Se realizó un pequeño Análisis Exploratorio de Datos para entender las variables y preparar el dataset antes de aplicar los modelos.
- **Ubicación**: Código disponible en `exploracion_datos.Rmd` (y su reporte generado `exploracion_datos.nb.html`).
- **Contenido**: 
  - Limpieza de datos.
  - Visualización de outliers y distribuciones.
  - Análisis de correlación y multicolinealidad (VIF).
